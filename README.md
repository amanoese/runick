yukichant
---
yukichantは、テキストデータを呪文に変換するコマンドです。

## Install

```bash
$ npm install -g yukichant
```

## Usage

```bash
$ chant
冠を御手を絢爛汚れ。我を意思に雫よ現れよ。仕打ち契約に羽根よ刻み。此処に剣に貫き。
```

## Develop

install dependency
```bash
$ npm install
```

execute command
```bash
## run encode
$ npm run dev unko
## run decode
$ npm run dev unko | npm run dev -- -d
```
